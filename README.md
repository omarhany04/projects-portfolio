# Omar Hany Projects Portfolio

Welcome to Omar Hany's projects portfolio.

This website is designed as a focused project index for quickly reviewing Omar's work from a CV, resume, or shared portfolio link.

## How To Use

Start from the **Featured Projects** section to see the strongest and most polished projects first. These cards include larger summaries, key features, technologies, live demo links, GitHub links, and project logos.

Scroll down to **Project Directory** to browse all listed projects in a compact format.

## Browse Projects

Use the category buttons in the **Project Directory** to filter projects by area, such as:

- Full Stack
- AI Product
- AI Frontend
- Machine Learning
- Backend API
- Security
- Embedded
- Desktop App

Choose **All** to return to the full list.

## Search

Use the search box to find projects by:

- project name
- category
- technology
- keyword from the description

For example, you can search for `React`, `Supabase`, `NLP`, `API`, `Arduino`, or `PostgreSQL`.

## Open Links

Each project may include:

- **Live Demo**: opens the deployed project or demo video
- **GitHub**: opens the source code repository

Links open in a new browser tab.

## Viewed Projects

After you click a project's **Live Demo** or **GitHub** link, a **green check mark** appears on that project card.

This helps you keep track of which projects you have already opened while reviewing the portfolio.
